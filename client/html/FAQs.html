Ask a question

When I open Chrome it says "Can't update chrome" what should I do?

You should reinstall chrome from our official website. It would not affect you settings.

How can I clear my browsing history?

Go to Settings → Privacy and security → Clear browsing data. You can choose what to delete and in what time range.

Why is Chrome using so much memory?

Although we do as much as possible to optimize our browser, a lot of open windows still can use much memory. Try closing tabs that you no longer use.

How do I import my settings from another browser?

To import you settings from another browser go to Bookmarks → Import bookmarks and settings. Then choose the browser and what you would like to import.

How do I set Chrome as my default browser?

Go to Chrome Settings → Default browser → Click "Make default". You may be prompted by your operating system to confirm.

Is browsing with Chrome secure?

Yes, Chrome includes built-in security features like Safe Browsing, sandboxing, and regular security updates.

Can I use Chrome extensions?

Yes, Chrome supports thousands of extensions available from the Chrome Web Store.

Does Chrome sync data across devices?

Yes, if you're signed into your Google account, Chrome can sync your bookmarks, passwords, history, and more across devices.

Is Google Chrome free to use?

Yes, Google Chrome is completely free to download and use.
